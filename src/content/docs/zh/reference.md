- MVP PRD：首个公开版本的功能范围。
- Wireframes：早期信息架构和页面流。
- Design tokens：颜色、字体、间距和视觉约束。
- Visual interaction spec：阅读器、列表、解释与设置页交互规格。
- Article action interaction spec：收藏、稍后读、已读、不感兴趣等状态契约。
